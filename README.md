# Swiper

Get orders made on an account on an exchange and duplicate them multiplied by a ratio.

## setup

Provide those env values:

```
FETCHER_EXCHANGE
FETCHER_API_KEY
FETCHER_API_SECRET
```

## Available exchange

- binance USDT futures (but dont copy orders made from android / iphone)
